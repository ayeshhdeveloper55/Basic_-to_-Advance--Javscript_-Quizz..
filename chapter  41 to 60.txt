Invoke Function ka Matlab

JavaScript me function ko invoke karna ka matlab hota hai function ko call karna — yani us function ka code run karwana.

Jab tum function banati ho, wo sirf ek definition hota hai.
Uska code tab tak execute nahi hota jab tak tum function ko call (invoke) na karo.

Example
function greet() {
  console.log("Hello, Ayesha!");
}

// Function invocation (calling)
greet();


function greet() → ye function ka definition hai.

greet(); → ye invocation hai. Is line se function ka code run hoga.

Function ko Invoke Karne ke Tareeqe

Directly call karna: myFunction();

Event ke through:

<button onclick="myFunction()">Click me</button>


Automatically call (IIFE — Immediately Invoked Function Expression):

(function(){
  console.log("I run immediately!");
})();

Important Point

Agar tum sirf function ka naam likho: greet (without ()), to wo execute nahi hoga, sirf uska reference milega.

Execute karne ke liye brackets () lagane zaroori hain.
